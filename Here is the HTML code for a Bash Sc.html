Bash Scripting Walk-through: Core Concepts

Objective: To create a single Bash script that demonstrates the use of variables, strings, metacharacters, loops, and conditional logic (Booleans).

Instructions: Follow the steps below. In your Linux environment, create a file named core_script.sh and execute the commands in your terminal as instructed.

Activity Tasks

  1. Setup and Shebang:

    Open your script file and add the shebang line to ensure it executes with Bash. Add a comment with your name and the date.

    #!/bin/bash
    # Student: [Your Name] - Date: [Current Date]

    Action: Save the file.

  2. Variables and Strings:

    Define two variables: USER_NAME for your name (as a string) and START_COUNT with a numerical value of 1. Use double quotes (") around your name to demonstrate proper string handling and the **metacharacter** for preserving spaces.

    USER_NAME="James Parriett"
    START_COUNT=1
    TARGET_COUNT=5
    echo "--- Script Started by: $USER_NAME ---"

    Action: Add the code above to your script and save.

  3. Loop (for loop) Implementation:

    Write a for loop that iterates from START_COUNT up to TARGET_COUNT (inclusive). In each iteration, print the current loop number. **Use single quotes (')** in the echo statement to observe the difference in metacharacter handling if you were to use a variable inside it (though not required here, it's a good learning point).

    for i in $(seq $START_COUNT $TARGET_COUNT); do
        echo "Current loop iteration: $i"
    done

    Action: Add the code above to your script and save.

  4. Boolean/Conditional Logic (If Statement):

    After the loop, use an if statement to check if the TARGET_COUNT is greater than 3. This condition will evaluate to a **Boolean** true or false (exit code 0 for true, non-zero for false). If the condition is true, print a success message. Use the metacharacter `>` (redirection) to write a brief summary to a new file named summary.txt.

    # Boolean/Conditional Logic Test
    if [ $TARGET_COUNT -gt 3 ]; then
        echo -e "\nCondition is TRUE: Target count ($TARGET_COUNT) is greater than 3."
        echo "Script completed successfully on $(date)." > summary.txt
        echo "Summary written to summary.txt"
    else
        echo "\nCondition is FALSE: Target count is NOT greater than 3."
    fi
    echo "--- Script Finished ---"

    Action: Add the conditional block to your script and save. The complete script should now be ready.

  5. Execution and Verification:

    Make the script executable, run it, and check the newly created file.

    chmod +x core_script.sh
    ./core_script.sh
    cat summary.txt
