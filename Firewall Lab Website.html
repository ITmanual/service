Windows Firewall Lab: Block Inbound and Outbound Ports 20 & 21

This lab demonstrates how to block inbound and outbound TCP ports 20 and 21 (FTP data and control) using Windows Defender Firewall on Windows 11. You’ll learn both GUI and PowerShell methods, verify the rules, and safely revert them.

Lab Objectives

  • Create inbound block rules for TCP ports 20 and 21.
  • Create outbound block rules for TCP ports 20 and 21.
  • Verify rules via GUI and PowerShell.
  • Revert or disable rules safely.

Part 1: Block inbound TCP ports 20 & 21 (GUI)

  1. Open Windows Security → Firewall & network protection → Advanced settings.
  2. Select Inbound Rules → New Rule….
  3. Choose Port → Next.
  4. Select TCP → Specific local ports → enter 20,21 → Next.
  5. Choose Block the connection → Next.
  6. Select all profiles (Domain, Private, Public) → Next.
  7. Name the rule Block Inbound TCP 20-21 (FTP) → Finish.

Part 2: Block outbound TCP ports 20 & 21 (GUI)

  1. Select Outbound Rules → New Rule….
  2. Choose Port → Next.
  3. Select TCP → Specific remote ports → enter 20,21 → Next.
  4. Choose Block the connection → Next.
  5. Select all profiles → Next.
  6. Name the rule Block Outbound TCP 20-21 (FTP) → Finish.

Part 3: Verify Rules

GUI Verification

  • Check Inbound Rules for Block Inbound TCP 20-21 (FTP).
  • Check Outbound Rules for Block Outbound TCP 20-21 (FTP).

PowerShell Verification

Get-NetFirewallRule -DisplayName "Block Inbound TCP 20-21 (FTP)" | Format-Table DisplayName, Enabled, Direction, Action
Get-NetFirewallRule -DisplayName "Block Outbound TCP 20-21 (FTP)" | Format-Table DisplayName, Enabled, Direction, Action

Part 4: Create Rules with PowerShell

# Block inbound TCP 20-21
New-NetFirewallRule `
  -DisplayName "Block Inbound TCP 20-21 (FTP)" `
  -Direction Inbound `
  -Action Block `
  -Protocol TCP `
  -LocalPort 20,21 `
  -Profile Domain,Private,Public `
  -Enabled True

# Block outbound TCP 20-21
New-NetFirewallRule `
  -DisplayName "Block Outbound TCP 20-21 (FTP)" `
  -Direction Outbound `
  -Action Block `
  -Protocol TCP `
  -RemotePort 20,21 `
  -Profile Domain,Private,Public `
  -Enabled True

Part 5: Reverting Rules

Disable or Remove (PowerShell)

Disable-NetFirewallRule -DisplayName "Block Inbound TCP 20-21 (FTP)"
Disable-NetFirewallRule -DisplayName "Block Outbound TCP 20-21 (FTP)"

Remove-NetFirewallRule -DisplayName "Block Inbound TCP 20-21 (FTP)"
Remove-NetFirewallRule -DisplayName "Block Outbound TCP 20-21 (FTP)"

End of lab. You have successfully blocked inbound and outbound FTP ports (20 & 21) on Windows 11 using Windows Defender Firewall.
