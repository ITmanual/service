This guided lab covers key Layer 2 concepts from Domain 2.0 (Network Implementation) and Domain 5.0 (Network Troubleshooting).

Domain 2.2: L2 Switching

  • STP vs. RSTP (FSTP): Standard STP (802.1D) convergence takes 30-50s. Rapid-STP / Fast STP (802.1w) converges in under 6 seconds.
  • Link Aggregation (LACP): Bundles multiple links into a single logical channel (Port-Channel) for higher bandwidth and fault tolerance.
  • Root Bridge Selection: Configured via bridge priority (e.g., lower values like 4096 take priority).

Domain 5.3: Troubleshooting

  • Switching Loops: Prevented dynamically by STP blocking redundant paths.
  • LACP Mode Mismatches: Ensure LACP `active` mode is configured symmetrically on connected peers to create Port-Channels.
